Stress TestingNon-functional✕ NoNone. Same as above.

Note on terminology: "Acceptance" and "UAT" below track closely by design — UAT is this project's actual (informal) implementation of acceptance testing. They're kept as separate columns because UAT is persona/journey-shaped while formal Acceptance Testing implies Gherkin/BDD-style specs, which this project doesn't use.
